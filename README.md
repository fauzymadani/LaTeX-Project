## Package
see the preamble in `main.tex` for what package i use. i'd reccomend installing `texlive-full` on your linux distro.
example:
```bash
sudo apt install texlive-full
```

## Preview
just run:
```bash
chmod +x compile.sh
./compile.sh
```
